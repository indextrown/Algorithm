## 누적합
누적합이란 요소들의 누적된 합의 의미로 어떠한 배열을 기반으로 앞에서부터 요소들의 누적된 합을 저장해 새로이 배열을 만들어서 이를 활용하는 것이다.
```swift
// prefixSum을 만들때 0인덱스는 안쓴다
// psum의 첫번째 요소(인덱스1)는 첫번째 까지 합한것...n번째 요소는 n번째 까지 합한 것
```

### 인덱스 규칙
```swift
// 문제의 인덱스(1-based)와 맞추기 위해
// 배열 앞에 더미 0을 추가한다
var arr = [0] + inputArray

```

### 누적합 배열 생성 공식
```swift
psum[i] = psum[i - 1] + arr[i]
```
- psum[0] = 0 (더미 값)
- psum[1] : 1번째 요소까지의 합
- psum[i] : i번째 요소까지의 합

### 구간 합 공식
```swift
// [a, b] 구간 합
psum[b] - psum[a - 1]
```
